# 📚 E-Learning -StudyHub (Web Application)

## 🔍 Project Summary

Study Hub is a UI-based e-learning web application, and this project focuses on automating its user interface functionalities using Selenium WebDriver with Java.
The automation framework is designed to simulate real user interactions on the web application and validate critical workflows to ensure functionality, usability, and reliability.

---

## 🎯 Functional Coverage

The following UI functionalities are automated:

* User Registration
* User Login
* Profile Update
* Contact Us – Form Submission and Validation 
* Course Navigation and Video Playback Validation

---

## 🛠️ Technologies Used

* **Programming Language:** Java
* **Automation Tool:** Selenium WebDriver 
* **Testing Framework:** TestNG
* **Build Tool:** Maven
* **Design Pattern:** Page Object Model (POM)
* **Version Control:** Git & GitHub

---

## 🧪 Automation Approach

* Performed **UI Testing** by automating browser actions using Selenium WebDriver
* Simulated real-time user behavior such as form submission, navigation, and data input
* Validated UI elements, page navigation, and application responses
* Used **TestNG** for structured test execution and assertions
* Implemented **Page Object Model (POM)** for better code organization and reusability
